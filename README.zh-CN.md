### 一切就绪之后

- #### 怎样开始开发?

  ##### 在终端里输入

  ```sh
  npm i -g pnpm && pnpx @resistjs/go
  ```

  若需要 **更新框架** , 请重新运行上面的命令. 这将在不影响到已有项目的前提下更新到当前版本。
